## 곡 배정

### 1인당 곡 개수

- 총 개수: 20곡
- 전부 투기타로 가정하여 20곡 * 2 / 6명 = 1인당 6곡 + 4파트
- 약 리드3곡 리듬3곡 + @
- 원기타 노래는 리드로 카운트

### 의견

- 원기타인 노래 중 투기타 가능한 노래는 세션 및 곡 추천자와 상의해서 투기타로 진행하고 싶습니다.
  - 투기타 가능 노래: 끝없이 우리는

### 배정 방법

- `곡추천자`는 해당 곡에 반드시 들어감
- `곡추천자`는 리드기타로 배정 (의견있을시 리듬으로 조정 가능)
- 원기타 노래는 리드로 카운트
- 곡의 개수가 정확하게 6명으로 나누어 떨어지지 않으니 일부 인원의 배정되는 노래수가 더 많지만 최대한 균등하게 배정 (예를들어 1인당 6곡이지만 남는곡을 가져가는데 7곡이 아닌 8곡을 배정하는 일은 없음)
- 리드 리듬도 역시 최대한 균등하게 배정 (1:1 비율)
- 우선순위는 `꼭 하고싶다` > `시키면한다` > `실력이안된다`
- 배정순서 역시 `꼭 하고싶다` > `시키면한다` > `실력이안된다` 순
- 득표수 상위 노래부터 다음과 같은 방법으로 배정
  1. `꼭 하고싶다`가 2명 혹은 1명일 경우 자동 배정
  2. 1번이 전부 완료된 후 6곡이 넘어가는 인원은 당사자가 개인 선호도에 따라 조절하여 곡 제외
  3. `꼭 하고싶다`가 3명 이상일 경우 배정된 곡이 적은 인원이 우선순위 배정 (동점일 경우 상의 후 배정)
  4. `꼭 하고싶다`가 전부 결정되면 `시키면한다`를 다시 1번 부터 동일한 방식으로 배정


- 리드 리듬 파트분배는 리드 리듬 개수 카운트 및 당사자간 상의 후 배정
- 곡 개수가 6개가 다 채워질 경우 그 인원은 곡 배정 제외
- 1인당 6곡이 다 채워진 이후 의견 조율하여 남는곡 추가 배분
